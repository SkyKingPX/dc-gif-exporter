# dc-gif-exporter

> [!IMPORTANT]
> This requires you to request a discord data export of your account data.
> 
> While there is some personal data about your account stored in the user.json file, the page is fully client side and no data is sent to any server. - Feel free to check for yourself.

This is a Discord GIF exporter designed to help you save all of your GIFs due to Tenor Shutting down on June 30th 2026.

Feel free to use the `Download All` function but be warned that **will** slow down your device for a while du to it opening a lage amount of tabs.

Read more on https://dc-gif-exporter.vercel.app/

Contributions and suggestions are welcome!
